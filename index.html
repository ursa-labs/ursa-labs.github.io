Shared Infrastructure for Data Science

We are working to improve the state of the art in open source software for data access, preparation, cleaning, analytics, feature engineering, and other essential tasks in the data science lifecycle. Rather than build tools for a single programming language like Python, We intend to develop portable computational libraries that can be used in many programming languages. Learn more about the Ursa Labs technology vision.

Partnering with Industry

Ursa Labs depends on financial support from companies that benefit from the advancement of open source data science software. It is our goal to bring together talented systems engineers to develop next-generation software for the public good. We also will directly collaborate on projects with engineers who work for other firms. Learn more about how you or your company can support Ursa Labs.

Powered by Apache Arrow™

Apache Arrow is a community-led, cross-language development platform for in-memory analytics, backed by leaders from many other popular open source data projects. We are driving Arrow development efforts to create shared next-generation computational infrastructure for data science. Apache Arrow empowers data science developers to seamlessly share data and algorithms across programming languages like R and Python.
